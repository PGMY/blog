なんとも試行錯誤しまくるハメになったものの、Octopress + GitHub Pages + Travisの通りにやってやっとこさうまく行きました。
はじめはトークンを利用しての方法にしてたんですがうまく行かず、結局公開鍵使ってのデプロイですー。

これをたたけばオッケーです。

上記URLからデプロイキーを登録します
